The Document's Role

A document is a work collected by provisioners or created by scribes.

The Document's Attributes

idUnique ID for the document task (not the document)
document_idUnique ID for the document
agreement_idUnique ID of the source agreement under which it was collected
source_guidThe unique ID of the document as assigned by the source, if available
titleThe title of the document
publication_tsPublication date of the document
formatFormat (mime-type or other attribute) of the document
encodingCharacter encoding of the document
languageHuman language in which the document is written
pathPath to persisted image of the document
errorsErrors related to the document itself (bad HTML for example)
source_document_id(Attribute in translated docs only) Document ID of the original
translations(Attribute in original documents only)
targetTarget document file form
document_idUnique ID of the translated document
