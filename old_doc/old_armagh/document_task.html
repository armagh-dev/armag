The Document Task's Role

The document task tracks the progress of an individual document through the scriptorium.

The Document Task's Attributes

idUnique ID for the document task (not the document)
document_idUnique ID for the original document
agreement_idUnique ID of the source agreement under which it was collected
pile_idUnique ID of the pile and pile task to which this document belongs
publication_tsPublication date of the document
provisioner_tsTime the document was collected by the provisioner
translationsOne of the following for each different target translation of this original:
targetTarget document file form
scribe_idPID or other unique ID of the scribe claiming the task
claim_tsTime the scribe claimed the translation task
started_tsTime after pulling the source document that the scribe started the translation
completed_tsTime the scribe completed the translation, before persisting
saved_tsTime the scribe completed saving the translated work
document_idUnique ID of the translated document
errorsList of errors encountered attempting to complete translation that are not document-related (resource failure, etc)
